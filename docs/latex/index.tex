\begin{DoxyEnumerate}
\item Register a new user.
\item Open the database insert a new row into {\ttfamily User\+Roles} table
\item Set the {\ttfamily User\+Id} to the {\ttfamily Id} of the user you want to make admin.
\item Set the {\ttfamily Role\+Name} to {\ttfamily Admin}.
\item Save the changes.
\item The user should now have admin access.
\item Admin tools can be accessed by clicking the new buttons in the navbar.
